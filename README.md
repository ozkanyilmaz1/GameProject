# GameProject
23 Ocak 2021 Ödev 5
Bu proje ile şunları hedeflemekteyiz: 

* Oyuncuları sisteme kayıt etmek
* Bilgilerini güncelleyebilmek
* Kayıtlarını silebilmek
* Kampanya eklemek
* Kampanya güncellemek
* Kampanya silmek gibi özellikleri de barındırıyoruz.

MERNİS doğrulaması da gerçekleştirerek bir kişinin birden fazla hesap oluşturup, çakışmasına engel olmayı sağlamaktayız. Bunun için kişiden "TcNo", "Ad", "Soyad", "DoğumYılı" bilgilerini istemekteyiz.
